Requirements
========================================================================================================
Must have IntelliJ IDE,
Ensure the Project JDK is set before running,
Use the provided "assignment3database.sql" file to create a student table with initial data in pgAdmin4


Steps to Compile and Run Application
=====================================
1. Open the project in IntelliJ
2. Ensure the Project JDK is set before proceeding
3. Locate and open the main.java file of the project
	- Change the url variable to match your database
	- Change the user variable to match your pgAdmin 4 username
	- Change the password variable to match your pgAdmin 4 password
4. In the Try block, you can find calls to the CRUD functions already available, but you may add your own
5. Once you are ready to run, click the run button next to the main method
